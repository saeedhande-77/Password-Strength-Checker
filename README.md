# 🔐 Password Strength Checker

## Overview

Password Strength Checker is a cybersecurity web application that helps users evaluate the security of their passwords. The tool analyzes password complexity and provides instant feedback to encourage stronger password practices.

## Features

* Real-time password strength analysis
* Strength categories:

  * Weak
  * Medium
  * Strong
* Visual strength indicator
* Password security recommendations
* Clean and responsive user interface

## Technologies Used

* HTML5
* CSS3
* JavaScript

## How It Works

The application checks the password for:

* Minimum length requirement
* Uppercase letters
* Lowercase letters
* Numbers
* Special characters

Based on these criteria, it calculates the password strength and displays appropriate feedback.

## Cybersecurity Relevance

Weak passwords are one of the most common causes of unauthorized account access. This project promotes cybersecurity awareness by encouraging users to create strong and secure passwords.
